learner – All-in-one interface¶

Learner is a wrapper class that regroups a :py:mod:Dataset, a :py:mod:ForwardModel, and an :py:mod:InverseModel. It is the easiest way to use models, and is flexible enough to be initialized from robots, Forward and InverseModel instances.

The class accept any from of iterable as input and ouput vector, as well as pandas.Series. In the latter case, the index should match either self.Mfeats or self.Sfeats.
